TELECHARGEMENT

MacOS Téléchargement Windows Téléchargement Linux Téléchargement Instructions Le jeu ne peut pas être joué sur mobile.
  • Attention : ce jeu est conçu pour être utilisé sur un ordinateur
    et nécessite un clavier pour pouvoir être joué.
    Téléchargement du jeu selon votre appareil.
    Sélectionnez le fichier correspondant à votre système.
  • Si vous utilisez un ordinateur MacOS, cliquez sur le bouton de
    téléchargement sous macOS. En ouvrant le fichier .zip, celui-ci se
    convertira automatiquement en une application. Il vous suffira de
    double-cliquer sur l’icône du jeu pour le lancer et commencer votre
    partie.
  • Si vous utilisez le système d'exploitation Windows, cliquez sur le
    bouton de téléchargement sous Windows. Après avoir double-cliqué
    sur le fichier .zip, un dossier nommé « Windows » apparaîtra dans
    vos téléchargement. Ouvrez-le, puis double-cliquez sur le fichier
    "wukongs_legend.app.exe" Le jeu se lancera immédiatement
    et vous pourrez en profiter.
  • ⁠Si vous utilisez Linux, cliquez sur le bouton de téléchargement sous
    Linux. Après le téléchargement, votre fichier se trouvera dans le
    dossier "Téléchargements" de votre ordinateur. Il s'agit d’un fichier .zip
    que vous pouvez l’ouvrir en double-cliquant dessus pour l’extraire.
