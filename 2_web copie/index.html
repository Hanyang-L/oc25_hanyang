La page d'introduction est constituée des éléments tels que la présentation du sujet, les objectifs, le choix du thème et les difficultés rencontrés.

Sur la deuxième page « Le jeu » tous les détails autour de la conception du jeu Wukong's Legend est présenté. plus précisément le moteur de créations « Godot » accompagnés d'illustrations et d’explications sur les personnages, les objets, la musique ainsi que les inspirations.

La dernière page présente les liens de téléchargement du jeu pour macOS, Windows et Linux. Ce choix s’explique par le fait qu’il s’agit des trois systèmes d’exploitation les plus utilisés. Les instructions d’installation sont détaillées pour chacun d’entre eux.
